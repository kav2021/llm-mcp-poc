\# LLM MCP POC



\## Overview

This project is a Proof of Concept (POC) demonstrating an MCP-style agent

that interacts with InfluxDB and Grafana to query and visualize sensor data.



\## Features

\- Query InfluxDB for sensor metrics (temperature, PPFD, voltage, etc.)

\- MCP-style agent logic for tool-based interactions

\- Grafana dashboard integration

\- Modular Python code structure



\## Tech Stack

\- Python

\- InfluxDB

\- Grafana

\- Git \& GitHub



\## How to Run

1\. Install Python 3.x

2\. Install dependencies

3\. Configure InfluxDB and Grafana credentials

4\. Run the agent script



\## Notes

\- Secrets are excluded using `.gitignore`

\- This repository is used for version control and collaboration
